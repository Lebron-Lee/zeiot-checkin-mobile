# 火山引擎新年活动配色分析

## 主色调
- 主背景：深红色渐变 #C8102E → #E8001D（中国红）
- 次背景：暖米色/象牙白 #F5E6C8（卡片区域）
- 主按钮：鲜红色 #E8001D 带粉红光晕
- 强调色：金黄色 #FFD700 / #F5A623（装饰、标题）
- 文字主色：深红褐 #8B1A1A / 白色 #FFFFFF
- 卡片背景：暖白 #FFF8F0

## 设计语言
- 顶部导航：深红色背景 + 白色文字
- Hero区：红色渐变天空 + 山水国画背景
- 按钮：圆角胶囊形，红色背景白色文字，带光晕
- 装饰：金色星星/烟花粒子效果
- 卡片：白色/米色背景，圆角，轻微阴影
- 整体：喜庆中国年风格，红+金+暖白三色体系

## 应用到我们系统的色彩方案
- 背景：深红 #8B0000 → 暗红 #C8102E 渐变
- 主按钮：鲜红 #E8001D / 金色 #FFD700
- 卡片：半透明暖白 rgba(255,248,240,0.1)
- 文字：白色 + 金色
- 装饰：金色粒子、红色光晕
- 大屏：深红背景 + 金色装饰 + 白色文字
